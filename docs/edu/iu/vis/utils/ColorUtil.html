AverageColor()method
public static function AverageColor(sourceBitmapData:BitmapData, clipRect:Rectangle = null):uint

Calculates the average hexidecimal color from a BitmapData instance.

Parameters
sourceBitmapData:BitmapData — A referenced BitmapData instance.
 
clipRect:Rectangle (default = null) — A Rectangle indicating the area from the referenced BitmapData to find the average color. If no Rectangle is provided, the entire BitmapData area is utilized.

Returns
uint — Returns a 16-bit hexidecimal color.

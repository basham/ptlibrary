GetBoundingRectFromPointRegion()method
public static function GetBoundingRectFromPointRegion(source:BitmapData, point:Point, fillColor:uint = 0xFFFFF000, uniqueColor:uint = 0xFFFFFF00, copySource:Boolean = false):Rectangle

Finds the bounding rectangular region of a blob containing the given point.

Note: The unique color must not be present in the sourceBitmapData, otherwise, false results are returned

Parameters
source:BitmapData — The referenced BitmapData.
 
point:Point — The point relative to the source BitmapData contained in a blob region.
 
fillColor:uint (default = 0xFFFFF000) — A 32-bit ARGB color filling the blob region containing the point.
 
uniqueColor:uint (default = 0xFFFFFF00) — A 32-bit ARGB color that is not present in the source BitmapData. If the color is not unique, the function will return false results.
 
copySource:Boolean (default = false) — A Boolean indicating if the function should use a working copy (true) or modify the source BitmapData (false).

Returns
Rectangle — Returns a Rectangle indicating the bounding region of a blob containing the given point.
GetSmartColorBoundsRect()method 
public static function GetSmartColorBoundsRect(source:BitmapData, rect:Rectangle, mask:uint, color:uint, relativeBounds:Boolean = false):Rectangle

Finds the color region within a rectangular bound in the source bitmap.

Parameters
source:BitmapData — The referenced BitmapData.
 
rect:Rectangle — The retangular bounds to search for a colored region.
 
mask:uint — A 32-bit ARGB color mask.
 
color:uint — A color to be searched for in the source BitmapData.
 
relativeBounds:Boolean (default = false) — A Boolean indicating how to adjust the returned bound value. If set to false (default), the bound coordinates are calculated relative to the rect parameter. If set to true, the bound coordinates are calculated relative to the source BitmapData bounding rectangle.

Returns
Rectangle — Returns a Rectangle indicating the bounds of a certain colored region.
SelectClosestColor()method 
public static function SelectClosestColor(color:uint, source:BitmapData, rect:Rectangle = null, operation:String = "<="):Point

Find a single point along a retangular bounds who's pixel color matches a given color according to a certain operation.

Parameters
color:uint — A 32-bit ARGB color attempted to be matched.
 
source:BitmapData — The referenced BitmapData.
 
rect:Rectangle (default = null) — The retangular bounds to search for the point.
 
operation:String (default = "<=") — Any standard set of operations in string form to compare the color to a point, e.g. '<=', '==', '>', '!=', etc.

Returns
Point — Returns a Point if a matched pixel color is found or NULL if no match is found.

TwoBitBitmap()method 
public static function TwoBitBitmap(source:BitmapData, ratio:Number = .5, colorLow:uint = 0xFF000000, colorHigh:uint = 0xFFFFFFFF, mask:uint = 0xFFFFFFFF):void

Reduces a BitmapData to two 32-bit colors according to a threshold.

Parameters
source:BitmapData — The referenced BitmapData.
 
ratio:Number (default = .5) — The relative percentage between the two high/low colors defining the color value to threshold the source BitmapData.
 
colorLow:uint (default = 0xFF000000) — The low, extreme 32-bit ARGB color for thresholding. Default: 100% visible black.
 
colorHigh:uint (default = 0xFFFFFFFF) — The high, extreme 32-bit ARGB color for thresholding. Default: 100% visible white.
 
mask:uint (default = 0xFFFFFFFF) — The 32-bit ARGB color mask for thresholding.

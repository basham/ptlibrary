Classpublic class Normalize

The Normalize class defines general, static utility functions to calculate an average value based on an array of numeric values. Averaging is considered the the mathematical equivalent of statistical normalizing.

Method detail
Average()method
public static function Average(values:Array, averageType:String, truncateExtremes:Boolean = false, truncatePercentage:Number = .25):Number

Calculates the average value of an array of numeric elements.

Parameters
values:Array — An array of numeric values.
 
averageType:String — An indicator dictating the type of averaging function to utilize (MEAN, LINEAR, QUADRATIC).
 
truncateExtremes:Boolean (default = false) — A Boolean flag indicating to automatically remove any extreme data in the values array.
 
truncatePercentage:Number (default = .25) — A Number percentage value (0 ... 1) of the number of extreme low and high elements in the values array to be removed from the array.

Returns
Number — Returns a calculated average value based on the elements in the values array.
Linear()method 
public static function Linear(values:Array):Number

Calculates the average linear-weighted value of an array of numeric elements. The larger an element's index, the less its value affects the calculated average. Each element's weight linearly decreases as index increases.

Parameters
values:Array — An array of numeric values.

Returns
Number — Returns a calculated average value based on the elements in the values array.

public static function Quadratic(values:Array):Number

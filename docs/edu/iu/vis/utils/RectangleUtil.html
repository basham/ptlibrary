ApproachingSquare()method 
public static function ApproachingSquare(rect:Rectangle, deviance:Number = 0):Boolean

Determines if a Rectangle's sides are of equivalent length within some given percentage of deviance.

Parameters
rect:Rectangle — The referenced Rectangle.
 
deviance:Number (default = 0) — A percentage (0 ... 1) indicating how close the sides' length must match. A value of 0 means the width and height must be equivalent. A value of .5 means a side could be up to 50% smaller or larger than the other side.

Returns
Boolean — Returns a Boolean.
Boundless()method 
public static function Boundless(rect:Rectangle, minRect:Rectangle = null, useMinArea:Boolean = false):Boolean

Determines if a Rectangle's dimensions (default) or surface area is beneath a given threshold.

Parameters
rect:Rectangle — A Rectangle to be compared.
 
minRect:Rectangle (default = null) — A Rectangle setting the threhold to compare rect. If not provided, minRect is assumed as a default Rectangle with zero width and hight.
 
useMinArea:Boolean (default = false) — If false (default), compares the two Rectangles according to dimensions. If true, compares the two Rectangles according to surface area.

Returns
Boolean — Returns a Boolean.
